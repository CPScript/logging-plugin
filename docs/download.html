How to Install and Use

Follow the steps below to install and use the Server Plugin:

  1. Download the latest version of the plugin from the Versions list above.
  2. Place the downloaded .jar file into the plugins directory of your Minecraft server.
  3. Restart your server to load the plugin.
  4. Configure the plugin by editing the config.yml file located in the plugin's folder.
  5. Join your server and enjoy the logging features!
